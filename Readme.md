
# Dungeon Shooters

2D shooter with RPG inspiration.


## Tech Stack

Typescript, ViteJS




## Run Locally

Clone the project

```bash
  git clone https://github.com/matheusluizn/Dungeon-Shooters
```

Go to the project directory

```bash
  cd Dungeon-Shooters
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
